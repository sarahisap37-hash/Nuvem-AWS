## 📑 Módulo 1: Introdução à Nuvem AWS (Fundamentos)
Focado no entendimento estratégico de por que as empresas migram para a nuvem.

* **Definição de Cloud Computing:** Entrega de recursos de TI sob demanda (computação, banco de dados, armazenamento) via internet com modelo de **pagamento conforme o uso**.
* **Vantagens da Nuvem:**
    * **Agilidade:** Rapidez para implantar recursos.
    * **Elasticidade:** Escalar recursos para cima ou para baixo automaticamente.
    * **Economia de Escala:** Preços menores devido ao alto volume de clientes AWS.
    * **Alcance Global:** Implantação em minutos em diversas regiões do mundo.
* **AWS Cloud Adoption Framework (AWS CAF):** * Organiza a jornada de migração em **6 perspectivas**: Negócios, Pessoas, Governança (Foco em Gestão) e Plataforma, Segurança, Operações (Foco Técnico).
