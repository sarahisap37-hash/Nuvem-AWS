# ☁️ AWS (Amazon Web Services)

##  O que é a AWS: Uma definição clara da plataforma.

De forma simples, a AWS é uma plataforma que aluga capacidade computacional, armazenamento e banco de dados pela internet. Em vez de uma empresa comprar e manter servidores físicos em sua própria sede, ela "aluga" esses recursos da Amazon, acessando tecnologia de ponta sob demanda.


##  Infraestrutura Global: Como ela funciona (Regiões e Zonas de Disponibilidade)

###  Regiões
As regiões são áreas geográficas onde a AWS possui seus data centers.  
Por exemplo, existem regiões na América do Sul, América do Norte, Europa, Ásia, entre outras.
Cada região funciona de forma independente, o que aumenta a segurança e a estabilidade.  
Além disso, escolher uma região mais próxima do usuário ajuda a melhorar a velocidade de acesso (menor latência).

###  Zonas de Disponibilidade (AZs)
Dentro de cada região existem várias zonas de disponibilidade, que são data centers separados fisicamente.
Essas zonas são projetadas para não depender umas das outras.  
Isso significa que:
- Se uma zona tiver algum problema (queda de energia, falha, etc.),  
- As outras continuam funcionando normalmente.

##  Economia da Nuvem: Como funciona o modelo de precificação (Pay-as-you-go) e como a empresa gera lucro através da economia de escala.

O modelo **pay-as-you-go** funciona cobrando apenas pelo que é utilizado. Ou seja, a empresa não precisa investir em infraestrutura própria e paga conforme o uso dos serviços, como tempo de servidor, armazenamento ou dados utilizados. Isso traz mais economia e flexibilidade, já que é possível aumentar ou reduzir os recursos a qualquer momento.

A AWS gera lucro através da economia de escala. Como atende milhões de clientes, consegue reduzir custos operacionais e oferecer preços mais baixos. Mesmo assim, o grande número de usuários e o alto consumo de serviços garantem um lucro elevado.
