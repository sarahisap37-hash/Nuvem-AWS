# ☁️ AWS (Amazon Web Services)

##  1- O que é a AWS
A AWS é uma plataforma que aluga capacidade computacional, armazenamento e banco de dados pela internet. Em vez de uma empresa comprar e manter servidores físicos em sua própria sede, ela "aluga" esses recursos da Amazon, acessando tecnologia de ponta sob demanda. 

##  2- Infraestrutura Global
A AWS possui uma infraestrutura distribuída pelo mundo inteiro, o que garante desempenho, segurança e alta disponibilidade dos serviços.

###  Regiões
As regiões são áreas geográficas onde a AWS possui seus data centers.  
Existem várias regiões espalhadas pelo mundo (como América do Sul, América do Norte, Europa, Ásia, etc.).
Cada região funciona de forma independente, o que aumenta a confiabilidade dos serviços e permite que os usuários escolham locais mais próximos para melhorar a velocidade de acesso.

###  Zonas de Disponibilidade 
Dentro de cada região existem várias zonas de disponibilidade, que são data centers separados fisicamente.
Essas zonas são independentes entre si, com energia, rede e sistemas próprios.  
Isso garante que, se uma zona apresentar falha, as outras continuem funcionando normalmente.


##  3- Economia da Nuvem
O modelo **pay-as-you-go** funciona cobrando apenas pelo que é utilizado. Ou seja, a empresa não precisa investir em infraestrutura própria e paga conforme o uso dos serviços, como tempo de servidor, armazenamento ou dados utilizados. Isso traz mais economia e flexibilidade, já que é possível aumentar ou reduzir os recursos a qualquer momento.
A AWS gera lucro através da economia de escala. Como atende milhões de clientes, consegue reduzir custos operacionais e oferecer preços mais baixos. Mesmo assim, o grande número de usuários e o alto consumo de serviços garantem um lucro elevado.
